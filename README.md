# homebrew-tap

Homebrew tap for [nuri-yoo](https://github.com/nuri-yoo) tools.

## Install

```sh
brew tap nuri-yoo/tap
brew install branch-watch
```

## Tools

| Tool | Description |
|------|-------------|
| [branch-watch](https://github.com/nuri-yoo/branch-watch) | Track branch and fork sync status across your GitHub repositories |
